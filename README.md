# Gestión Agrícola 2.0

Plataforma web para la gestión integral de explotaciones agrícolas, maquinaria y facturación.

## Características
* **Roles de usuario:** Administrador, Maquinista y Agricultor.
* **GIS Integrado:** Visor de mapas con Leaflet para dibujar y visualizar parcelas.
* **Gestión de Tareas:** Asignación de trabajos y control de estados (Pendiente -> En Curso -> Finalizado).
* **Facturación:** Generación automática de facturas y control de pagos.

## Tecnologías
* PHP 7/8
* MySQL (Base de datos relacional)
* JavaScript (Leaflet.js para mapas)
* HTML5 / CSS3

## Instalación
1. Importar el archivo `database.sql` en phpMyAdmin.
2. Configurar `conexion.php` si es necesario.
3. Abrir en servidor local (XAMPP).
